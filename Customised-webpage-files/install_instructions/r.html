Newcastle University Managed Computers

The University provides 'packaged' software, which can be installed without admin rights on your managed laptop or workstation.

  1. Open 'Software Center' from the start button and search for 'CRAN-R'.
  2. Choose Git (x64) and click 'install'
  3. After install is completed, "R" can be launched from the start menu.
If you can't see the software you need, log a ticket at https://nuservice.ncl.ac.uk to request that it is added to Software Center. You'll need to provide the PC name.
